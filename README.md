# Grafos em C++

Projeto desenvolvido durante os estudos de Estruturas de Dados utilizando Grafos Direcionados e algoritmos clássicos de busca.

## Funcionalidades

* Criação de grafos direcionados
* Inserção de arestas com peso
* Verificação de conectividade
* Busca em Profundidade (DFS)
* Busca em Largura (BFS)
* Detecção de ciclos
* Verificação de alcançabilidade
* Problema de Rede (Network)
* Problema do Dominó

## Conceitos Trabalhados

* Grafos
* Lista de Adjacência
* DFS
* BFS
* Teoria dos Grafos
* Conectividade
* Ciclos em Grafos
* Caminhos
* Grafos Eulerianos

## Estrutura do Projeto

### graph.h / graph.cpp

Implementação da estrutura principal do grafo:

* Criação de vértices
* Inserção de arestas
* Grau do vértice
* Vizinhos
* Pesos
* Conectividade

### Busca em Grafos

* DFS (Depth First Search)
* BFS (Breadth First Search)
* Verificação de alcançabilidade
* Detecção de ciclos

### network.cpp

Análise de sobrecarga de rede utilizando grafos ponderados.

### domino.cpp

Verificação de possibilidade de montagem de dominó através de conceitos de grafos Eulerianos.
